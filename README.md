# 🎮 Tic Tac Toe

A browser-based Tic Tac Toe game built with **HTML**, **CSS**, and **JavaScript**. The game features winner detection, turn tracking, a Michael Jackson "Hee Hee" sound effect on each move, and a Frieren victory GIF.

## Features

* Classic two-player Tic Tac Toe
* Winner detection
* Turn indicator
* Reset game functionality
* Michael Jackson "Hee Hee" sound effect on every move
* Frieren victory GIF displayed when a player wins
* Responsive game board using CSS Grid

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)

## Getting Started

Clone the repository:

```bash
git clone https://github.com/Muhammadkhan001/tic-tac-toe.git
```

Then open `index.html` in your web browser.
